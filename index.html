Comandos

  • HTML
    • <!DOCTYPE html>: Define o tipo de documento e a versão do HTML.
    • <html>: Elemento raiz de uma página HTML.
    • <head>: Contém metadados/meta-informações da página HTML.
    • <meta charset="utf-8">: Define o conjunto de caracteres para a página.
    • <title>: Define o título da página, que é exibido na barra de título do navegador.
    • <body>: Contém o conteúdo visível da página HTML.
    • <main>: Representa o conteúdo principal ou importante na página HTML.
    • <h1> a <h3>: Representam cabeçalhos. <h1> é o cabeçalho de nível mais alto e <h6> é o mais baixo.
    • <section>: Define uma seção em um documento.
    • <!--...-->: Define um comentário.
    • <p>: Define um parágrafo.
    • <a>: Define um hiperlink.
    • <img>: Usado para incorporar uma imagem na página HTML.
    • <ul>: Define uma lista não ordenada (com marcadores).
    • <li>: Define um item da lista.
    • <figure>: Especifica conteúdo independente, como ilustrações, diagramas, fotos, trechos de código, etc.
    • <figcaption>: Define uma legenda para um elemento <figure>.
    • <ol>: Define uma lista ordenada (numerada).
    • <form>: Define um formulário HTML para entrada do usuário.
    • <fieldset>: É usado para agrupar elementos relacionados em um formulário.
    • <legend>: Define uma legenda para o elemento <fieldset>.
    • <label>: Define uma etiqueta para um elemento <input>.
    • <input>: Usado para criar controles de entrada interativos para formulários baseados na web.
    • <button>: Representa um botão clicável.
    • <footer>: Define um rodapé para um documento ou seção.
    • <em>: Representa ênfase no texto.
    • <strong>: Define texto importante.
  • CSS
    • body { }: Seleciona todo o corpo do documento.
    • h1 { }: Seleciona todos os elementos h1.
    • h2 { }: Seleciona todos os elementos h2.
    • .class-name { }: Seleciona todos os elementos com class="name".
    • h1, h2, p { }: Seleciona todos os elementos h1, h2 e p.
    • img { }: Seleciona todos os elementos de imagem.
    • hr { }: Seleciona todos os elementos hr.
    • footer { }: Seleciona todos os elementos footer.
    • a { }: Seleciona todos os elementos a.
    • a:visited { }: Seleciona todos os links visitados.
    • a:hover { }: Seleciona os links quando você passa o mouse sobre eles.
    • a:active { }: Seleciona o link ativo.
